# 영어 수업 자동 배정 기능 보고서

## 개요

Excel의 영어 수업 정보를 Firebase 학생에게 자동 배정하는 기능입니다.

## 주요 기능

### 1. 수업 관리 연동
- `classes` 컬렉션에 등록된 영어 수업과 자동 매칭
- `classId` 연결로 수업 관리와 동기화

### 2. 수동 매칭 UI ✨ (신규)
매칭 실패 수업에 대해 드롭다운으로 수업 관리의 기존 수업과 연결 가능

## 테스트 결과

| 항목 | 결과 |
|------|------|
| 수업 관리 영어 수업 | 37개 |
| 자동 매칭 | 58건 |
| 수동 매칭 필요 | 3건 (LE1, LE6, LT1) |
| 수동 매칭 후 | 70건 (LT1 → LT1b 연결 시) |

## 사용 방법

1. **학생 관리** → **"영어 배정"** 버튼 클릭
2. 자동 매칭 현황 확인
3. **수동 매칭**: 매칭 실패 수업에 드롭다운으로 수업 선택
4. **"배정 실행"** 버튼 클릭
