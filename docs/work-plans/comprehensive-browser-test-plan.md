## 1. 테스트 환경 준비

### 1.1 필수 계정 준비

| 역할 | 이메일 | 목적 |
|------|--------|------|
| Master | `master@example.com` | 모든 권한 테스트 |
| Admin | `admin@example.com` | 관리자 권한 테스트 |
| Math Teacher | `math@example.com` | 수학 강사 권한 테스트 |
| English Teacher | `english@example.com` | 영어 강사 권한 테스트 |
| Normal User | `user@example.com` | 일반 사용자 권한 테스트 |

### 1.2 테스트 데이터 준비

- [ ] 학생: 최소 20명 (active 15명, withdrawn 3명, prospect 2명)
- [ ] 반: 최소 10개 (수학 5개, 영어 5개)
- [ ] 강사: 최소 6명 (수학 3명, 영어 3명)
- [ ] 부서: 최소 5개 (다양한 카테고리)
- [ ] 일정: 최소 15개 (다양한 부서/년도)

### 1.3 브라우저 환경

- [ ] Chrome (최신 버전)
- [ ] Firefox (최신 버전)
- [ ] Safari (macOS)
- [ ] Edge (최신 버전)
- [ ] Mobile Safari (iOS)
- [ ] Mobile Chrome (Android)

### 1.4 개발 서버 실행

```bash
npm run dev
# http://localhost:5173 에서 확인
```
