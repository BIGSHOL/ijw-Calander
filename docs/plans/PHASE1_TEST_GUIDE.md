# Phase 1 수동 테스트 가이드

## 🎯 테스트 목적
UserDetailModal 컴포넌트가 정상적으로 분리되었는지 확인

## 🚀 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

---

## ✅ 테스트 시나리오

### 1. 설정 모달 접근
- [ ] 우측 상단 설정 아이콘 클릭
- [ ] SettingsModal 정상 표시
- [ ] "Permissions" 탭 선택
- [ ] "Users" 서브탭 선택

### 2. UserDetailModal 열기
- [ ] 승인된 사용자 목록에서 아무 사용자나 클릭
- [ ] UserDetailModal이 오버레이로 표시됨
- [ ] 사용자 이메일이 제목으로 표시
- [ ] 프로필 아이콘 정상 표시 (Master/Admin/Normal User)

### 3. 호칭 및 역할 변경
- [ ] "호칭 입력" 필드에 텍스트 입력 가능
- [ ] Master 로그인: 역할 드롭다운 표시 (Admin, Normal User 등)
- [ ] 역할 변경 후 즉시 반영됨
- [ ] 상태 드롭다운 (승인됨/대기중/차단됨) 동작

### 4. 강사 프로필 연동
- [ ] "강사 프로필 연동" 섹션 표시 (청록색 배경)
- [ ] 드롭다운에서 강사 선택 가능
- [ ] "연동 안함" 선택 가능
- [ ] 선택 후 체크 메시지 표시

### 5. 부서별 권한 관리
- [ ] 모든 부서 목록 표시 (색상 점 포함)
- [ ] 각 부서별로 "차단/조회/수정" 버튼 3개 표시
- [ ] 버튼 클릭 시 색상 변경 (회색/파란색/초록색)
- [ ] "전체 조회" 버튼: 모든 부서가 "조회"로 변경
- [ ] "전체 수정" 버튼: 모든 부서가 "수정"으로 변경
- [ ] "전체 차단" 버튼: 모든 부서 권한 제거
- [ ] "초기화" 버튼: 원래 상태로 복원

### 6. 사용자 삭제
- [ ] **본인 계정**: 삭제 버튼 표시 안 됨
- [ ] **다른 사용자**: "사용자 삭제" 버튼 표시 (빨간색)
- [ ] 삭제 버튼 클릭 시 확인 다이얼로그 표시
- [ ] 확인 후 사용자가 목록에서 사라짐
- [ ] 모달 자동 닫힘

### 7. 모달 닫기
- [ ] **X 버튼**: 우측 상단 X 클릭 시 모달 닫힘
- [ ] **외부 클릭**: 모달 바깥 영역 클릭 시 닫힘
- [ ] **ESC 키**: ESC 키 누르면 모달 닫힘
- [ ] **확인 버튼**: 하단 "확인" 버튼 클릭 시 닫힘

### 8. 권한 체크 (중요!)
**Master 계정으로 테스트**:
- [ ] 모든 사용자 역할 변경 가능 (다른 Master 제외)
- [ ] 모든 사용자 삭제 가능 (다른 Master 제외)
- [ ] 모든 부서 권한 수정 가능

**Admin 계정으로 테스트** (있다면):
- [ ] 역할 변경 드롭다운 비활성화
- [ ] Master 사용자 삭제 불가
- [ ] 본인 부서 권한만 수정 가능

### 9. 브라우저 콘솔 확인
- [ ] 콘솔에 에러 메시지 없음
- [ ] 경고(Warning) 없음
- [ ] React DevTools로 컴포넌트 트리 확인
